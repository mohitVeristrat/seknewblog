== Description ==

Automatically add InShare buttons to your posts. It works for single pages and loops.
The plugin features an options page in which the following can be configured:

Display options:

* Toggle hiding of button in loops 

Display style:

* Show button on top or bottom of post
* Choose button display style (counter top,counter right,default)
* Add breaks before and/or after the button
* Button preview
* Custom styling option, adds a containing div with classname which you can style in your stylesheet

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip and move the `linkedin-InShare` folder to your `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. After activation you can configure the plugin using the `settings` link or the menu
